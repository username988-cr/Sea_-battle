Морской бой на Python

Это консольная версия игры Морской бой на Python. Игрок и бот имеют поля произвольного размера, корабли размещаются автоматически или вручную. Реализована система сохранения, загрузки, логов и проверка победы.

Основные возможности

Выбор размера поля перед началом игры

Автоматическое или ручное размещение кораблей

Умное масштабирование кораблей под размер поля

Реалистическая проверка близости кораблей (нельзя ставить рядом)

Сохранение и загрузка игры

Бот атакует случайно и пытается победить

Ведение лога победителя

Игровой процесс

Игрок и бот получают свои поля

Игрок выбирает: загрузить сохраненную игру или начать новую

Игрок может разместить корабли вручную или автоматически

Игрок и бот ходят по очереди

Отмечаются попадания и промахи

После каждого хода происходит автосохранение

Победитель записывается в log.txt

Символы на поле

[ _ ] пустая клетка
[ # ] корабль
[ % ] подбитая часть корабля
[ @ ] промах

Управление

Простые команды в консоли:

i и j координаты выстрела
A сохранить игру
П выход из игры
